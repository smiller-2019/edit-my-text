# edit-my-text

An application to allow the creation of notes with or without an internet connection.

## Description

Allows the user to create notes on the fly such as code snippets and save them to the browser whether online or offline. The notes are still available after closing the app and opening it up again. The application can also be downloaded and installed.

## Installation

Type in npm install.
